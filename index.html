Sight Finder

Trabalho de IPM 2018/19

Descrição

A nossa web application é essencialmente um planeador de visitas a uma dada cidade. Os utilizadores vão poder encontrar os locais a visitar numa cidade. Os utilizadores vão poder ver os pontos a visitar em duas áreas: numa página com um mapa onde é possível ver exatamente onde são os pontos de interesse ou numa página com uma lista dos pontos de interesse que satisfazem as exigências dos utilizadores.

Será possível criar itinerários de visitas (e ver itinerários já criados?) e a aplicação indicará aos utilizadores os melhores transportes públicos para se deslocarem entre os diferentes pontos turísticos.

Vai existir também uma pequena página onde os utilizadores vão poder ver algumas informações sobre os pontos de interesse: horas de abertura e encerramento, rating, informações históricas, fotos partilhadas, etc.
